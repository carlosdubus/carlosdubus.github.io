About Me

I'm from Dominican Republic. I'm currently a Software Engineer student at RIT. I started coding at age of 14 when I created a ROMs/emulators website in PHP. In college, I moved to C/C++ and design patterns. I learned a lot on how to design modular code. Before finishing college, I became a freelancer at rentacoder.com (now freelancer.com) and then moved to odesk.com (now upwork.com). You can see my UpWork profile here. One of my clients became a long term client (Sonreign Media). In that position I worked with PHP, MySQL and Javascript mostly. I also was involved in the deployment and configuration of the website. (http://www.praisecharts.com).

I have experience working with many programming languages such as PHP, C, C#, Java, Python, Javascript. Almost all my focus right now is in web application development. I have experience working with SPA applications using Node.js, express, mongodb, and angular at the front end. Also, I have worked with MVC frameworks such as laravel, zend (php) and django (python).

I know how to work with source control systems such as SVN and Git and I know the importance of unit tests, dependency injection and modular code. I've used PHPUnit and mocha for unit tests and I've worked with Selenium for system tests.

I love working on new projects and learning new technologies!
